README

Title: Alpha Ocelot Scheduling

Description: This application was created to allow users to work with customer and appointment information
stored within a database. Users are able to view, create, update and delete customers and appointments within
the application. Users can can also view reports and other information about appointments and customers.

Author: Amador Montiel
Contact Info: amont99@wgu.edu
Application Version: 1.0
Date: 04/24/2021

IDE: IntelliJ Community 2020.3.3
JDK/SDK: Java 11.0.9, JavaFX SDK 11.0.2



How to run the program:
Upon launching the application, a login screen will appear. Once logged in, the user will be brought to the main application window.

From the main application window, the user will be able to add/modify/delete both customers and appointments. When selecting to add or modify customers or
appointments, the user is brought to new form screens to fill out the info needed. Also from the main window, the user can open different screens to view all
appointments and look at various reports that have been generated.

Additional Report: On the reports screen, an additional report that shows all appointments associated with a customer has been generated. The user selects from the combo-box the customer they want to see
appointments for.

MySQL Connector Driver: mysql-connector-java-8.0.22
